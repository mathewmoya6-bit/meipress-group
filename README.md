## 🛠️ Tech Stack

- Supabase (Authentication & Database)
- Font Awesome Icons
- Google Fonts (Inter)

## 🔧 Setup
1. Set up Supabase project
2. Run the migration in `supabase/migrations/init.sql`
3. Update `.env` with your values
4. Deploy to Vercel/Netlify
